# Аналитика "Энергосбыт"

Компания "Энергосбыт" обслуживает многоквартирные дома.  
В каждой квартире фиксируется потребление электроэнергии через умные счётчики

Бизнес хочет перейти от сырых данных к аналитическим витринам в формате, понятном руководству и операционным подразделениям.
Каждая ваша группа — это отдельная команда аналитики, отвечающая за свой кусок бизнес-аналитики.

Инженеры деляться на **4 группы**, которые будет анализировать разные бизнес-кейсы

1. [Аналитика трендов потребления во времени](./01/)
2. [Аналитика энергоэффективности и площадей](./02/)
3. [Аналитика качества показаний](./03/)
4. [Аналитика активности жильцов](./04/)
